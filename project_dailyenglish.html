Daily English is an animated learning module designed for middle school students, showcasing historical events that occurred on each date with beautifully crafted animations. As the Design Director, I designed the product structure and visuals, and led the team in creating over 1,000 animated courseware modules.

Project Background and Goals

Led by Professor Yang Yonglin from Tsinghua University's School of Foreign Languages, this educational project aims to make daily English learning more interesting and meaningful by creating a high-quality program with instructional videos for middle school students.

User Flow

Before we continued designing the app, I started by creating a user flow. It helps me to oversee the ideal flow for our app.

UI Design

Daily English is available on multiple platforms. For a better viewing experience, we recommend using a tablet.

Illustration & Animation

The project invited professional science writers to write the scripts. Based on these scripts, I collected real historical images and created beautiful illustrations in a unified style. Animations were made for each illustration using After Effects.
